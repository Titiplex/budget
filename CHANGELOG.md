# Budget v1.0.0

First stable local-first desktop release.

## Included

- Accounts, categories and transactions
- Internal transfers
- Budgets by category
- Recurring transactions
- Period reports and markdown export
- JSON backup / restore
- CSV import / export
- Tax report helper for France and Québec / Canada
- Desktop packaging flow
- Electron E2E smoke test coverage

## Notes

Budget is local-first. Data is stored locally with SQLite.
JSON backup is the canonical backup format.
